What is BiLinear Enlargement?

BiLinear enlargement is where linear interpolation is used to calculate what value a certain pixel should have. It achieves this by finding the 4 pixels in the original which surround the ideal point to get the colour information for that pixel. First horizontally it performs linear interpolation between the corner pixels which are in line, it takes the values from each of these at the correct x value. Then it performs a linear interpolation between these two values and takes the value at the correct y value, this is the value stored in the final image. How does this tool implement it?

The tool calculates what the ideal X and Y coordinate to look back at is, this is achieved by using the horizontal and vertical scale factors. It then takes the four pixels surrounding this ideal point, with topLeft being the topLeft pixel of the four etc. LeftX, RightX correspond to the x coordinates of the left and right pixels, and TopY, Bottom Y corrrespond to the y coordinates of the top and bottom pixel.
