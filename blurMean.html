What is a Mean Blur?

A Mean Blur is the simplest type of blur it just takes the arithmetic mean of the surrouding pixels and store's that vlaue at the current pixel. It looks at the pixels in a square surrounding the pixel we are finding the value of this makes it look unnatural so people tend to dislike it but it is very fast to run because it looks in a square. Mean blurs because of this are generally used for preprossesing when remvoing noise rather than when people will actually see the result of the blur.

How does this tool implement it?

To implement a mean blur this tool uses summed area tables, this enables each pixel's value to be calculated by drawing 4 values from the table, where as competeting soloutions the amount of operations increases per pixel as the size of the kernel increases. Once it has calculated the value from the summed area table then it divides that value by how many pixels are contained within that region.

A summed area table (integral image) keeps stored in each location the total sum from the top left of the array to that point.

How to use the Mean Blur in this tool

The Radius/Standard Deviation textbox for mean blur is the radius of the square, the width of the square will be two times the radius plus one.
